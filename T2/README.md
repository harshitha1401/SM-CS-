# ⌨️ Keyboard Event Monitor

## Overview

Keyboard Event Monitor is an educational web application built using Node.js, Express.js, HTML, CSS, and JavaScript.

The application demonstrates how browser keyboard events work by detecting and displaying key presses while the user interacts with the webpage.

It is designed for learning JavaScript event handling and frontend-backend communication.

---

## Features

- Detects Key Down events
- Detects Key Up events
- Displays pressed key
- Displays keyboard code
- Shows event timestamp
- Counts total events
- Modern responsive UI
- Built using Express.js
